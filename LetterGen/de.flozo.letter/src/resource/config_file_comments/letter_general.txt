File info:
This is the config file for general letter settings.

Format info:
Please note also the general format info in "master.config".
Keys containing "on", "hide", or "show" require boolean values "true" or "false".
If a value is left empty, a default value is used automatically.
Backslashes have to be escaped.
LaTeX commands can be used where characters/strings are to be specified.
LaTeX math symbols have to be surrounded with dollar signs.
